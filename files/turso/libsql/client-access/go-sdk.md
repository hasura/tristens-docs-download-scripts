# Go SDK

The Go SDK is implemented as a driver for the standard Go[ database/sql
package ](https://pkg.go.dev/database/sql). You can read more about how to install it and open a connection to
your Turso database using the documentation in the[ README on GitHub ](https://github.com/libsql/libsql-client-go/#readme). Note that
you must provide a[ client authentication token ](https://docs.turso.tech/reference/turso-cli#authentication-tokens-for-client-access)in the query string of the URL
you receive from the Turso CLI.
